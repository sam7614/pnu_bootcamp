# 부산대학교 부트캠프 훈련생 관리 웹앱

부산대학교 부트캠프의 7개 트랙과 교과형/몰입형 이수체계를 기준으로 학생 수강이력, 상담 상태, 다음 과정 추천을 관리하는 정적 웹앱입니다.

## 실행

별도 설치 없이 `index.html`을 브라우저에서 열면 실행됩니다.

## 주요 기능

- 대시보드: 첫 화면에서 중급과정/고급과정 현황을 분리하고, 각 과정 안에서 교과형과 몰입형을 별도 집계
- 트랙 상세 팝업: 대시보드의 트랙 카드를 클릭하면 세부 트랙별 학생 수, 상담 우선 대상, 교과형/몰입형 이수율과 과목 목록을 확인
- 고급과정 유형 팝업: 고급과정 카드의 교과형 또는 몰입형 영역을 클릭하면 해당 유형의 모듈별 이수 현황과 이수중인 학생 목록을 확인
- 팝업 학생 현황: 팝업의 인원 카드를 클릭하면 관련 학생 목록이 아래에 펼쳐지고, 교과형 학생 현황은 이수한 교과목과 이수해야 할 교과목을 구분해 표시
- 학생별 추천 펼침: 학생 이름을 클릭하면 해당 학생 아래에 중급/고급 교과형·몰입형 추천 트랙, 추천 근거, 트랙 설명, 다음 이수 과목, 교수 상담 포인트를 표시
- 중급과정: `코어AI트랙`은 클라우드·피지컬AI, 생성형AI, XR공간컴퓨팅, AI융합보안 4개 모듈로 관리하며, 융합AI트랙은 중급 교과형 없이 딥러닝 PBL·데이터분석 PBL 2개 몰입형 과목으로 관리
- 고급과정: 고급형 몰입형은 클라우드 AI PBL, 온디바이스 AI PBL, 로보틱스 AI PBL, 생성형 AI PBL, XR PBL, AI융합보안 PBL, 산업AI PBL 7개 과목으로 관리
- 관리자 학생관리: 100명 샘플 학생 데이터, 학생 검색, 상담 상태 필터, 신규 학생 등록, 상담 메모 저장, CSV 내보내기
- 음성 상담이력: 학생별 상담 음성파일 메타데이터와 전사문을 업로드하고 상담 요약/다음 행동/키워드로 정리
- 수강이력 관리: 학생별 교과형/몰입형 모듈 이수 여부 체크, 연도별·학기별 이수 내역 확인
- 트랙맵: 첨부된 이수체계도 이미지와 7개 트랙별 초급/중급/고급/PBL 모듈 확인
- 추천: 학생의 이수 모듈, 전공, 관심분야를 바탕으로 트랙과 몰입형 PBL 추천

## 데이터 저장

학생 데이터, 수강이력, 상담이력은 브라우저 `localStorage`에 저장됩니다. 수강이력에는 이수 연도와 학기 정보가 함께 저장됩니다. 음성파일은 브라우저 보안상 파일명/형식/크기 메타데이터만 저장하고, 상담 요약은 입력한 전사문 또는 상담 내용으로 생성합니다. 관리자 화면의 `초기화` 버튼을 누르면 100명 데모 데이터로 되돌릴 수 있습니다.
