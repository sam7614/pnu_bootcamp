# 04. 주요 화면 사용법

## 1. 운영 대시보드

운영 대시보드는 웹앱의 첫 화면입니다.

이 화면에서는 전체 현황을 빠르게 확인합니다.

### 주요 카드

- 관리 학생: 현재 조건에 해당하는 학생 수
- 평균 이수율: 학생들의 트랙 이수 진행률 평균
- 상담 우선: 위험도 높음 또는 관찰 필요 학생 수
- PBL 추천 가능: 몰입형 PBL 안내가 가능한 학생 수

각 카드를 클릭하면 아래에 관련 학생 목록이 펼쳐집니다.

### 필터

대시보드에는 다음 필터가 있습니다.

- 전체 연도 또는 특정 연도
- 전체 학기 또는 특정 학기
- 전체 과정 또는 특정 과정

필터를 바꾸면 대시보드 숫자와 학생 목록이 함께 바뀝니다.

## 2. 중급과정 대시보드

중급과정 대시보드는 코어AI트랙 중심으로 표시됩니다.

확인할 수 있는 내용은 다음과 같습니다.

- 코어AI트랙 전체 진행 현황
- 교과형 이수율
- 몰입형 PBL 현황
- 진행 중 학생 수
- PBL 추천 가능 학생 수
- 상담 필요 학생 수

## 3. 고급과정 대시보드

고급과정 대시보드는 트랙 카드를 한 줄에 3개씩, 총 2줄로 정리해 보여줍니다.

각 카드에서 확인할 수 있는 내용은 다음과 같습니다.

- 트랙명
- 트랙 설명
- 교과형 이수율
- 몰입형 PBL 이수율
- 관련 교과목 목록
- PBL 과목
- 진행 중 학생 수
- 전체 학생 수
- PBL 추천 가능 학생 수
- 상담 필요 학생 수

## 4. 학생관리 화면

학생관리 화면은 학생을 한 명씩 상담하고 관리하는 화면입니다.

### 주요 기능

- 학생 목록 확인
- 학생 검색
- 상담 상태 필터
- 학생 선택
- 상담 카드 확인
- 상담 메모 작성
- 상담 완료 처리
- 성적표 업로드 결과 확인

학생을 클릭하면 오른쪽 상담 카드에 해당 학생의 정보가 나타납니다.

## 5. 수강이력 화면

수강이력 화면은 학생별로 어떤 과목을 이수했는지 확인하는 화면입니다.

### 표시 방식

- 교과형 이수 모듈
  - 초급
  - 중급
  - 고급
- 몰입형 PBL 이력
  - 중급형
  - 고급형
- 연도별 이수 내역
- 학기별 이수 내역

이 화면에서는 이수한 과목과 이수해야 할 과목을 구분해서 볼 수 있습니다.

## 6. 트랙맵 화면

트랙맵 화면은 부트캠프 트랙별 이수체계도를 확인하는 화면입니다.

첨부된 이수체계도 이미지를 기준으로 트랙 구조를 이해할 수 있습니다.

## 7. 추천 화면

추천 화면은 학생별 추천 트랙과 추천 이유를 확인하는 화면입니다.

추천은 다음 요소를 바탕으로 구성됩니다.

- 학년
- 전공
- 관심 분야
- 목표 트랙
- 이수한 교과목
- 미이수 교과목
- PBL 진입 가능성

## 8. 검색창

상단 검색창에서는 다음 내용을 검색할 수 있습니다.

- 학생명
- 학과
- 교과목
- 트랙
- PBL 과목

검색 결과는 관련 자료를 빠르게 찾아가기 위한 보조 기능입니다.
