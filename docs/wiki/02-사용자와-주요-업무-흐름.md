# 02. 사용자와 주요 업무 흐름

## 주요 사용자

### 교수자

교수자는 학생 상담을 준비하고 진행하는 사용자입니다.

주요 업무는 다음과 같습니다.

- 학생별 이수 현황 확인
- 학생에게 다음 교과목 또는 PBL 추천
- 상담 메모 작성
- 상담 완료 처리
- 성적증명서 업로드 결과 확인

### 부트캠프 운영 관리자

운영 관리자는 전체 학생과 과정 운영 현황을 보는 사용자입니다.

주요 업무는 다음과 같습니다.

- 전체 관리 학생 수 확인
- 트랙별 이수율 확인
- 상담 우선 학생 확인
- PBL 추천 가능 학생 확인
- 학생 데이터 검색과 관리
- CSV 내보내기

### 교육과정 기획자

교육과정 기획자는 트랙별 학생 분포와 이수율을 보고 다음 과정 운영을 판단하는 사용자입니다.

주요 업무는 다음과 같습니다.

- 어떤 트랙에 학생이 몰리는지 확인
- 이수율이 낮은 트랙 파악
- 교과형과 몰입형 PBL 운영 계획 수립
- 신규 교과목 또는 PBL 개설 검토

## 대표 업무 흐름 1. 대시보드로 전체 현황 확인

1. 웹앱에 접속합니다.
2. 첫 화면의 운영 대시보드를 봅니다.
3. 관리 학생, 평균 이수율, 상담 우선, PBL 추천 가능 숫자를 확인합니다.
4. 필요한 경우 각 카드를 클릭해 관련 학생 목록을 펼칩니다.
5. 연도, 학기, 과정 필터를 바꿔 특정 조건의 현황을 확인합니다.

## 대표 업무 흐름 2. 학생 상담 준비

1. 왼쪽 메뉴에서 `학생관리`를 클릭합니다.
2. 학생명, 학과, 관심 분야 등으로 학생을 검색합니다.
3. 학생을 선택합니다.
4. 오른쪽 상담 카드에서 기본정보, 위험도, 상담 상태, 다음 상담일을 확인합니다.
5. 학생 이름 또는 추천 영역을 확인해 추천 가능한 트랙과 추천 이유를 봅니다.
6. 상담 메모를 작성하거나 수정합니다.

## 대표 업무 흐름 3. 수강이력 확인

1. 왼쪽 메뉴에서 `수강이력`을 클릭합니다.
2. 학생을 선택합니다.
3. 교과형 이수 모듈을 초급, 중급, 고급으로 확인합니다.
4. 몰입형 PBL 이력을 별도로 확인합니다.
5. 연도별, 학기별 이수 내역을 확인합니다.
6. 미이수 교과목을 바탕으로 다음 수강 계획을 상담합니다.

## 대표 업무 흐름 4. 성적증명서 업로드

1. 왼쪽 메뉴 하단의 `성적증명서 업로드` 버튼을 누릅니다.
2. PDF 파일을 선택합니다.
3. 팝업에서 학생 일반정보를 확인합니다.
4. 매핑된 교과목과 추천 가능 트랙을 확인합니다.
5. 정보가 맞으면 `확인`을 누릅니다.
6. 잘못된 내용이 있으면 `수정`을 누르고 정보를 고친 뒤 저장합니다.
7. 저장하지 않으려면 `취소`를 누릅니다.

## 대표 업무 흐름 5. 트랙과 과목 설명 확인

1. 대시보드에서 트랙명을 클릭합니다.
2. 트랙 개요와 목표를 확인합니다.
3. 교과목명을 클릭하면 수업계획서 형식의 안내를 확인합니다.
4. PBL 과목명을 클릭하면 과정 안내와 참여 혜택을 확인합니다.

## 상담자가 기억해야 할 원칙

- 추천 결과는 상담을 돕기 위한 근거입니다. 최종 판단은 교수자와 학생 상담을 통해 결정합니다.
- 성적증명서 매핑 결과는 자동 후보입니다. 저장 전 반드시 팝업에서 확인해야 합니다.
- 상담 메모에는 민감한 개인정보를 과도하게 적지 않는 것이 좋습니다.
- 현재 버전의 데이터는 브라우저 저장 방식입니다. 여러 PC에서 동일한 데이터가 자동 공유되는 실서비스 구조는 아닙니다.
